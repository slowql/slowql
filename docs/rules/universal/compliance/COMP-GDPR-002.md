# Potential Cross-Border Data Transfer (COMP-GDPR-002)

**Dimension**: Compliance
**Severity**: Medium
**Scope**: Universal (All Dialects)

## Description
Transferring personal data to foreign servers in non-adequate countries without SCCs or BCRs violates GDPR Chapter V and can result in significant fines.

**Rationale:**
Documentation for this rule's rationale is pending.

## Remediation / Fix
Document all cross-border data flows in your data inventory. Ensure Standard Contractual Clauses or Binding Corporate Rules are in place. Prefer data minimization — transfer only pseudonymized or anonymized data.
